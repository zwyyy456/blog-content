## Solution
Easy

## Code
```cpp
class Solution {
public:
    bool squareIsWhite(string coordinates) {
        if ((coordinates[0] - 'a' + 1 - coordinates[1]) % 2 == 0)
            return false;
        else
            return true;
    }
};
```
